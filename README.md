## 项目

### 项目目录整体划分

- app 核心目录 ，控制器、model 等都在这里

- bootstrap:  辅助函数

- config:  配置加载

- pkg:  配置加载service

- public:  静态文件

- resources: 资源文件/前端项目

- .air.toml:  监听

- .env: 配置

- .env.example: 辅助配置参考

### 前端后台

    # 参考至
    git clone https://github.com/vueComponent/ant-design-vue-pro.git
    
目录地址: `resource/front`
    
编译:
    
    # 项目根目录
    
    npm install
    npm run server
    
### 前端客户端

目录地址: `resource/views`
